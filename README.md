# 🔗 Сокращатель ссылок на Huma

> ⚠️ **Учебный проект / Песочница**
> Этот репозиторий — **учебный прототип**, созданный для изучения фреймворка [Huma](https://huma.rocks/) в связке с Chi и Valkey. Проект **НЕ предназначен для продакшена**. Используйте его как пример или отправную точку для собственных экспериментов.

## 🎯 Цель проекта

Проект создан для получения практического опыта работы с:
- **[Huma v2](https://github.com/danielgtaylor/huma)** — современный OpenAPI-first фреймворк для REST API на Go
- **[Chi](https://github.com/go-chi/chi)** — лёгкий и идиоматичный HTTP-роутер
- **[Valkey](https://valkey.io/)** — in-memory хранилище, совместимое с Redis (используется как кэш с TTL)
- **[Sonic](https://github.com/bytedance/sonic)** — высокопроизводительная JSON-библиотека

## ✨ Возможности

- Создание коротких ссылок (`POST /api/v1/shorten`)
- Получение метаданных ссылки по ID (`GET /api/v1/info/{id}`)
- Редирект на оригинальный URL (`GET /{id}` → 307 Temporary Redirect)
- Автоматическая генерация документации OpenAPI 3.x через Huma
- Кэширование в памяти с **TTL 5 минут** (Valkey)
- Быстрая сериализация JSON через Sonic

## 📡 Эндпоинты API

| Метод | Путь               | Описание                     | Тег       |
|-------|--------------------|------------------------------|-----------|
| POST  | `/api/v1/shorten`  | Создать короткую ссылку      | Links     |
| GET   | `/api/v1/info/{id}`| Получить информацию по ID    | Links     |

## ⚙️ Технологический стек

| Компонент       | Технология                            | Зачем                                    |
|-----------------|---------------------------------------|------------------------------------------|
| API-фреймворк   | [Huma v2](https://huma.rocks/)       | OpenAPI-first, авто-валидация, документация |
| Роутер          | [Chi v5](https://go-chi.io/)         | Лёгкий, совместимый со stdlib            |
| Кэш / Хранилище | [Valkey](https://valkey.io/)         | Совместимость с Redis, изучение новой экосистемы |
| JSON            | [Sonic](https://github.com/bytedance/sonic) | Высокопроизводительный маршалинг    |
| Язык            | Go                                    |                                          |

## ⚠️ Известные ограничения

- Нет аутентификации / rate limiting
- Нет персистентного хранения (данные теряются при остановке Valkey)
- TTL захардкожен на 5 минут
- Нет тестов
